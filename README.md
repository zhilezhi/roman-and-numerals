# roman-and-numerals
> This is a tool for converting between Roman numerals and Arabic numerals.
> 这是一个罗马数字和阿拉伯数字互相转换的工具.
## 如何使用
```
npm i roman-and-numerals
```
or
```
yarn add roman-and-numerals
```
